# Lumora — Responsive Landing Page

A responsive landing page built for a fictional AI workspace product called **Lumora**, created as part of a frontend internship assignment.

## 🔗 Live Demo
lumora-landing-page1.netlify.app

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript

## ✨ Features
- Fully responsive design (mobile, tablet, desktop)
- Animated hero section with floating cards
- Auto-scrolling logo strip
- Features grid with visual mockups
- How it works — 3-step section
- Testimonials section
- Interactive pricing toggle (Monthly / Annual)
- Smooth scroll reveal animations
- Mobile hamburger menu
- Clean dark theme with accent colors

## 📁 Project Structure
landing/
├── index.html    # Main HTML structure
├── style.css     # All styles & responsive breakpoints
└── script.js     # Interactivity & animations
